python 实现余弦相似度过滤豆瓣租房小组中介贴

前几天做了个微信找房机器人, 爬取豆瓣租房小组的时候发现很多伪装的中介帖，如下这种格式：

可月付 无中介 方庄地铁附近 芳城园一区单间出租

可月付 无中介 方庄地铁附近 芳城园一区主卧次卧出租

方庄地铁附近 芳城园一区次卧出租
